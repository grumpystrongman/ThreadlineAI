## Longer About copy

ThreadlineAI is a Windows-native AI sidecar for high-context work. It attaches beside the user's active app, tab, document, terminal, or workflow; previews approved context; routes Ask through a local provider bridge; and turns work into summaries, handoffs, decisions, risks, and next actions.

The product direction is local-first, visible, provider-agnostic, and privacy-aware. It is built for users who need AI help while working across many sources without constantly copying and pasting context into a separate chat window.

## One-paragraph external description

ThreadlineAI is a Windows-first AI companion that follows approved work context across desktop apps, browser tabs, documents, terminals, and sessions. It runs through a local service, keeps context visible before provider calls, supports Work Thread memory, and helps turn scattered work into summaries, handoffs, decisions, risks, and next actions. It is being shaped as commercial Windows software with service lifecycle, diagnostics, clear-data controls, release validation, and privacy-first defaults.

## Tone guidance

Use plain language. Do not overstate maturity. The product is commercially shaped and worth trying, but it is still actively developing. The best positioning is serious pre-release product foundation, not finished enterprise SKU.
